# Learnify Project

**Факултетен номер:** 2301321032

**Име на проекта:** Learnify – Online Learning Platform  

## Описание
Проектът включва база данни за онлайн платформа за курсове, диаграми на модела и Power BI доклад.  

## Структура на проекта
- `Learnify_Database/` – SQL файлове за функции, тригери, процедури и данни.
- `Learnify_Model_Diagrams/` – Диаграми (Chen, Crow's Foot, UML Physical и UML Data Warehouse).
- `Learnify_Power_BI/` – Power BI доклад с  5 визуализации.

## Инструкции за стартиране
1. Заредете `.sql` файловете от `Learnify_Database/` във вашата SQL Server база данни (LearnifyDB).
2. Прегледайте Power BI отчета (`Learnify_Power_BI.pdf`), който съдържа всички визуализации (5 страници) за анализа на данните.
3. Прегледайте визуализациите.

## Използвани технологии
- Microsoft SQL Server
- draw.io (за диаграми)
- Power BI (за визуализации)
- GitHub (за съхранение)
